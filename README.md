# juniortest
Задание для PHP разработчика Junior уровня с deadline на 3 дня:  Задача: Вы должны создавать формы и просматривать с «чистым» PHP и CSS. Форма должна содержать поля фамилии, имени, дня рождения и номера телефона. После сохранения полей формы вы должны записать эти данные в таблицу клиентов в базе данных. Вы должны отобразить все данные в виде столбцов. Номер телефона должен быть доступен для поиска.
